# Getting Started

This section covers the basics of getting a Raspberry Pi up and running.

## Prerequisites

- A Raspberry Pi (any model)
- A microSD card (16 GB or larger recommended)
- A power supply appropriate for your Pi model
- [Raspberry Pi Imager](https://www.raspberrypi.com/software/) installed on your computer

## Installing an Operating System

1. Download and install [Raspberry Pi Imager](https://www.raspberrypi.com/software/).
2. Insert your microSD card into your computer.
3. Open Raspberry Pi Imager, select your Pi model, choose an OS, and select your SD card.
4. Click **Write** and wait for the process to complete.

## First Boot

1. Insert the microSD card into your Pi.
2. Connect peripherals (keyboard, monitor) or configure headless access.
3. Power on the Pi.
4. Follow the on-screen setup wizard (if using Raspberry Pi OS Desktop).

!!! tip
    For headless setups, enable SSH by placing an empty file named `ssh` in the boot partition before first boot.
