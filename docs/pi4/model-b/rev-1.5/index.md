## Identification

- **Revision codes:** b03115 (2GB), c03115 (4GB), d03115 (8GB)
- **Release:** Mid-2021 onwards
- **Key changes from Rev 1.4:** BCM2711 C0 stepping, default clock raised to 1.8GHz, improved thermal performance
